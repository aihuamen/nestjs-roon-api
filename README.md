# NestJS Roon API

## Description

[Nest](https://github.com/nestjs/nest) framework with [Roon API](https://github.com/RoonLabs/node-roon-api) (and [Discord RPC](https://github.com/discord/discord-rpc)).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
